```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│              "단절(斷絶) → 연결(連結)"                            │
```

> 우리는 **세대 간, 국경 간 연결**을 만들어 이 문제에 대응한다."

# 3. 솔루션 후보 상세 설계

## 솔루션 A: "연결 다리" — 세대 연결 플랫폼

### 컨셉
초등학생과 독거 어르신을 1:1로 매칭하여 주 1회 영상통화하는 프로그램.
아이는 "사회적 관계"를, 어르신은 "안부 확인 + 존재감"을 얻는다.

### 왜 이것인가?
| 문제 | 이 솔루션이 해결하는 것 |
|------|----------------------|
| 아이의 공감 부족 → 따돌림 | 다른 세대와 대화하며 공감 능력 발달 |
| 어르신의 고독 → 고독사 | 정기적 안부 확인 + 감정적 연결 |
| 세대 간 단절 | 직접 대화를 통한 상호 이해 |

### 실행 방법
| 단계 | 내용 | 기간 |
|------|------|------|
| 1. 프로토타입 | 우리 팀 주변 초등학생 5명 + 독거 어르신 5명 매칭 | 1개월 |
| 2. 파일럿 | 주 1회 15분 영상통화 (우리가 중간에서 관리) | 2개월 |
| 3. 기록 | 전후 설문, 인터뷰, 변화 기록 | 상시 |
| 4. 확산 | 결과를 정리하여 지자체/복지관에 프로그램 제안 | 4개월 |

### 필요한 것
- 협력 기관: 지역 아동센터 1곳 + 독거노인 복지관 1곳
- 도구: 태블릿 (어르신용), 줌/카카오 영상통화
- 인력: 우리 팀이 매주 코디네이터 역할

### 일본 벤치마킹
- **Aoi Care 모델** (후지사와): 어린이와 노인이 함께 차를 만들고 판매
- **치매카페**: 아이들이 치매 어르신과 교류하는 프로그램 (10년간 6,000명 참여)

## 솔루션 B: "오늘의 안부" — AI 안부 확인 서비스

### 컨셉
독거 어르신에게 매일 아침 카카오톡 메시지를 보내고, 답이 없으면 봉사자/복지사에게 알림을 보내는 간단한 시스템.

### 왜 이것인가?
| 한국 현실 | 일본 벤치마킹 |
|-----------|-------------|
| 고독사 3,924명, 매년 증가 | IoT 모니터링 (전기/수도 패턴 감지) |
| 독거노인 23.7% | 외로움·고립 대책법 (2024) |
| "발견까지 며칠" 문제 | 모션센서, 봉사자 순찰 |

### 실행 방법
| 단계 | 내용 | 기간 |
|------|------|------|
| 1. 카카오톡 챗봇 개발 | 매일 아침 "오늘 기분이 어떠세요?" 메시지 | 2주 |
| 2. 답변 패턴 분석 | 미응답 시 → 담당자에게 알림 | 2주 |
| 3. 파일럿 운영 | 독거 어르신 10명 대상 테스트 | 2개월 |
| 4. 복지관 연계 | 기존 돌봄 체계와 연동 | 3-4개월 |

### 기술 스택
- 카카오톡 채널 + 챗봇 API (무료)
- 미응답 알림 → 슬랙/카카오 알림톡
- 대시보드: 간단한 웹페이지로 현황 확인

### 차별점
- 일본의 센서 기반 모니터링은 비용이 높음 → 우리는 **이미 어르신이 쓰는 카카오톡**을 활용
- "감시"가 아닌 **"대화"** 형태 → 어르신이 부담 없이 참여

## 솔루션 C: "도망쳐도 괜찮아" — 학교폭력 피해학생 정보 허브

### 컨셉
학교폭력을 당했을 때 "어디로 가야 하는지" 한눈에 볼 수 있는 웹사이트.
일본 프리스쿨 사례 + 한국 대안교육 정보 + 신고 방법 + 상담 연결.

### 왜 이것인가?
| 현실 | 문제점 |
|------|--------|
| 학폭 피해율 2.5% (역대최고) | 피해 후 "뭘 해야 하는지" 모름 |
| 대안교육기관 267개 | 정보가 흩어져 있음, 접근성 낮음 |
| Wee클래스, 꿈드림 센터 | 존재를 모르는 학생이 대다수 |
| 일본 프리스쿨 출석 인정 | 한국은 아직 미비 → 인식 개선 필요 |

### 웹사이트 구성

```
┌──────────────────────────────────────────────────────┐
│  "도망쳐도 괜찮아" — 학교가 전부가 아니야             │
├──────────────────────────────────────────────────────┤
│                                                      │
│  [1] 지금 도움이 필요해                              │
│      → 117 신고, Wee센터, 경찰, 상담 전화번호        │
│      → 버튼 하나로 바로 전화 연결                    │
│                                                      │
│  [2] 학교 말고 다른 곳이 있어                        │
│      → 전국 대안교육기관 267개 지도                   │
│      → 홈스쿨링 시작하는 법                          │
│      → 검정고시 준비 방법                            │
│                                                      │
│  [3] 일본은 이렇게 해                                │
│      → 프리스쿨 사례, 교육기회확보법 소개            │
│      → "학교에 안 가도 괜찮다"는 인식 변화           │
│                                                      │
│  [4] 같은 경험을 한 사람들의 이야기                  │
│      → 익명 경험담 공유 게시판                       │
│      → 한일 청소년 교류 커뮤니티                     │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 실행 방법
| 단계 | 내용 | 기간 |
|------|------|------|
| 1. 정보 수집 | 전국 대안교육기관, 상담기관 DB 구축 | 2주 |
| 2. 웹사이트 제작 | 모바일 최적화, 10대가 쓰기 쉬운 UI | 3주 |
| 3. 콘텐츠 제작 | 일본 사례 소개, 경험담 수집 | 상시 |
| 4. 바이럴 | 인스타그램, 에브리타임, 학교 게시판 | 2-4개월 |

## 솔루션 D: "한일 연결 프로젝트" — 공동 다큐/리포트

### 컨셉
일본 방문 시 현지 대학생·활동가와 함께 한일 공통 문제(따돌림/고령화)에 대한 공동 리포트 또는 짧은 다큐멘터리를 제작.

### 왜 이것인가?
- 일본 방문 기회를 최대한 활용
- "벤치마킹"을 넘어 "공동 대응"의 첫걸음
- 결과물이 남아서 이후에도 영향력 발휘

### 실행 방법
| 단계 | 내용 | 기간 |
|------|------|------|
| 1. 사전 연결 | 일본 대학생/NPO와 온라인 미팅 | 방문 전 |
| 2. 현지 취재 | 프리스쿨, 치매카페, 지역포괄지원센터 방문 & 인터뷰 | 방문 중 |
| 3. 공동 제작 | 한일 양국 관점을 담은 10분 영상 또는 보고서 | 방문 후 2주 |
| 4. 공유 | 유튜브 + SNS + 펠로우십 발표 | 4개월 |

### 취재 대상 (일본)

| 기관/장소 | 소재지 | 취재 포인트 |
|-----------|--------|------------|
| 도쿄 슈레 (프리스쿨) | 도쿄 | 등교거부 대안의 선구자 |
| D카페 (치매카페) | 도쿄·오사카 | 세대 교류 현장 |
| 지역포괄지원센터 | 전국 | 돌봄 통합 시스템 |
| 오무타시 치매케어 모델 | 후쿠오카 | 국제적으로 인정받은 모델 |

## 솔루션 E: "세대연결 보드게임" — 공감을 게임으로

### 컨셉
학교폭력, 고독, 세대 갈등 상황을 체험하는 보드게임을 제작.
초등학교와 경로당에서 함께 플레이 → 자연스러운 세대 교류 + 공감 교육.

### 왜 보드게임인가?
- 교육적이면서도 진입장벽 낮음
- 제작 비용 상대적으로 낮음 (종이 프로토타입 가능)
- 실물이 남아서 지속 사용 가능
- 초등학교·경로당에 배포 가능

### 게임 설계 방향

```
[게임명] "하루를 걸어봐" (한일 2개국어 버전)

[컨셉]
플레이어가 유년기~노년기를 걸어가며
각 시기에 닥치는 상황 카드를 뽑고 선택함.
→ 따돌림 상황, 진로 고민, 외국인 친구, 독거 생활 등
→ "나라면 어떻게 했을까?" 대화 유도

[게임 메커니즘]
- 4인 플레이 (이상적: 2명 청소년 + 2명 어르신)
- 협력형: 경쟁이 아니라 함께 문제를 해결
- 감정 카드: 상황에서 느낀 감정을 공유
- 일본 사례 카드: 일본에서는 이렇게 했다는 정보

[제작 계획]
1단계: 종이 프로토타입 (2주)
2단계: 테스트 플레이 3회+ (2주)
3단계: 디자인 보완, 소량 인쇄 (50세트) (1개월)
4단계: 초등학교·경로당 시범 운영 (1-2개월)
```

# 5. 일본 방문 시 할 수 있는 것

## 방문 전 준비

| # | 할 일 | 방법 |
|---|-------|------|
| 1 | 방문처 사전 섭외 | 이메일/X(트위터) DM으로 연락 |
| 2 | 질문지 준비 | 한국 상황 설명 + "어떻게 시작했는지" 중심 |
| 3 | 일본 대학생 파트너 찾기 | 대학 한일교류 서클, SNS |
| 4 | 촬영 장비 | 스마트폰 + 짐벌 + 마이크 (충분) |

## 방문 중 체크리스트

| 방문처 | 볼 것 | 물어볼 것 |
|--------|-------|-----------|
| **프리스쿨** (예: 도쿄 슈레) | 교육 방식, 학생 분위기, 시설 | "처음에 학부모·학교의 저항은?" |
| **치매카페** | 운영 방식, 참여자 반응, 세대 교류 | "초기 자원봉사자 모집은?" |

## 방문 후

# 7. 게임을 활용한 인지 훈련 프로그램

## 7.1 고스톱과 치매 예방: 근거 정리

### 학술적 사실

| 질문 | 답 |
|------|---|
| 고스톱 자체를 연구한 논문이 있나? | **없다** — 고스톱 대상 임상연구는 존재하지 않음 |
| 카드·보드게임의 효과는? | **있다** — 프랑스 20년 추적: 보드게임 → 치매 위험 15%↓ |
| 마작은? | **있다** — 단기 기억·주의력·논리력 향상 (태극권보다 효과적) |
| 핵심 메커니즘은? | 게임 자체보다 **"사람과 만나서 대화하며 하는 행위"**가 핵심 |

### 고스톱의 인지 훈련 요소

| 요소 | 고스톱에서 | 인지 기능 |
|------|-----------|-----------|
| 계산 | 점수 계산 | 수리력 |
| 기억 | 나온 패 기억 | 단기 기억 |
| 전략 | 고/스톱 판단 | 실행 기능 |
| 사회적 교류 | 떠들며 즐김 | **이게 가장 중요** |

### 문제: 도박 연상

고스톱 = 돈 → 요양시설에서 "도박" 이미지 → 보호자·기관 반발 가능

**→ 해결: 고스톱 메커니즘은 살리되, 도박성은 제거한 새 게임 설계**

---

## 7.2 비도박 게임 설계

### 게임 A: "기억 고스톱" — 가장 추천

| 항목 | 내용 |
|------|------|
| **규칙** | 일반 고스톱과 동일하게 진행 |
| **추가 규칙** | 패를 낼 때마다 "그 달과 관련된 기억" 하나 말하기 |
| **예시** | 8월 패 → "8월이면 추석이지~" / 4월 패 → "우리 아들 4월에 태어났어" |
| **보너스 점수** | 기억 공유 +1점, 공감 리액션 +1점, 가장 재미있는 이야기 = "이야기왕" |
| **돈 대신** | 칭찬 스티커, 간식 쿠폰, "이야기왕" 타이틀 |
| **장점** | 새 규칙 안 배워도 됨 / 회상치료 자연 결합 / 승패보다 이야기가 중심 |

### 게임 B: "사계절 여행" — 화투 활용 신규 게임

| 항목 | 내용 |
|------|------|
| **컨셉** | 화투 12달 그림 = 여행지. 패를 맞추면 "그 달의 여행지에 도착" |
| **승리 조건** | 가장 많은 여행지를 모으면 승리 |
| **고스톱과 같은 점** | 같은 화투 패, 짝 맞추기 메커니즘, 기억력+전략 |
| **고스톱과 다른 점** | 고/스톱 없음, 점수="추억 포인트"(돈 아님), 협력 모드 가능 |
| **대화 유도** | 3월(벚꽃) 맞추면 → "할머니, 벚꽃 보러 가신 적 있으세요?" |

### 게임 C: "화투 짝 맞추기" — 가장 단순 (치매 초기용)

| 항목 | 내용 |
|------|------|
| **규칙** | 화투 패를 뒤집어 놓고 메모리 게임 (2장씩 뒤집어 짝 맞추기) |
| **장점** | 규칙 극도로 단순, 화투 패 그대로 사용, 기억력 훈련 극대화 |
| **대상** | 인지 수준 낮은 분도 가능 |
| **도박성** | 전혀 없음 |

## 7.3 침대에서 할 수 있는 방식

### 방식 1: "중계형" — 봉사자가 손이 되어주기 (비용 0원)

```
병실 가운데 테이블에 고스톱판

🧓A (침대) → "나 7월 내!"
                  ↓
          🙋 봉사자가 대신 패를 냄
                  ↓
🧓B (침대) → "에이~ 뺏겼네!"

= 어르신은 "두뇌 + 대화"만 하면 됨
= 봉사자(쉬었음 청년)와의 세대 교류 보너스
```

### 방식 2: "태블릿형" — 화상통화 + 터치 게임

| 필요한 것 | 태블릿 + 침대 거치대 (~2만원) |
|-----------|-----|
| 앱 | 기존 고스톱 앱 멀티플레이 + 영상통화 화면 분할 |
| 핵심 | **상대 얼굴이 보여야** 사회적 효과 있음 |
| 장점 | 다른 병실, 다른 시설, 집에 있는 사람과도 가능 |

### 방식 3: "라디오형" — 음성만으로 (시력·거동 어려운 분)

| 방법 | 진행자가 패 상황을 말로 설명, 어르신이 음성으로 선택 |
|------|-----|
| 장점 | 눈 안 보이는 분, 손 못 쓰는 분도 참여 가능 |
| 효과 | 기억력 + 판단력 = 인지 훈련 극대화 |

### 방식 4: "자석 보드형" — 실물 패 (상체 움직임 가능한 분)

| 방법 | 오버베드 테이블에 자석 보드 + 대형 화투 패 (일반의 2배) |
|------|-----|
| 제작 비용 | ~3만원 (자석시트 + 대형 화투 인쇄) |
| 장점 | 기울어져도 패 안 떨어짐, 실물의 촉감 |

# 8. 쉬었음 청년 × 노인 돌봄 연결 모델

## 8.1 왜 "쉬었음 청년"인가?

### 현황

| 지표 | 수치 |
|------|------|
| 쉬었음 청년 | **48.5만명** (2026.2) |
| 니트(NEET) 전체 | 125만명 (15-29세 중 18.3%) |
| 20대 후반 니트 비중 | **20.2%** (가장 높음) |
| OECD 비교 | 한국만 2014년 대비 증가 (다른 나라 모두 감소) |

### "쉬었음 청년" + "독거 어르신" = 서로가 서로의 해답

```
쉬었음 청년의 문제           독거 어르신의 문제
─────────────────           ─────────────────
"갈 곳이 없다"              "올 사람이 없다"
"할 일이 없다"              "해줄 사람이 없다"
"나를 필요로 하는            "나를 기억하는
 곳이 없다"                  사람이 없다"
         │                         │
         └─────────┬───────────────┘
                   ▼
         서로를 연결하면
         양쪽 문제가 동시에 풀린다
```

### 왜 일반 봉사자가 아닌 쉬었음 청년인가?

| 일반 봉사자 | 쉬었음 청년 |
|------------|-----------|
| 시간 내기 어려움 | **시간이 있다** (핵심 자원) |
| 단발성 참여 | **지속 가능** |
| 동기 = 스펙 | 동기 = **사회 복귀 디딤돌** (진정성) |
| 에너지 분산 | 이 활동에 **집중** 가능 |

## 8.2 연결 모델 5가지

### 모델 1: "동네 손주" — 정기 방문 매칭

| 항목 | 내용 |
|------|------|
| 구조 | 청년 1명 ↔ 어르신 1-2명, 주 2회 방문 |
| 활동 | 대화, 장보기 동행, 병원 동행, 스마트폰 교육 |
| 청년에게 | 활동비 월 30~50만원 + 사회서비스 경력 |
| 어르신에게 | 정기 안부 + 감정적 연결 + 실질적 도움 |
| 벤치마킹 | 일본 **후레아이 키푸**: 돌봄 시간 적립 → 나중에 본인이 사용하는 "시간은행" |

### 모델 2: "디지털 도우미" — IT 교육

| 항목 | 내용 |
|------|------|
| 구조 | 요양시설/경로당 방문, 주 1-2회 |
| 활동 | 스마트폰, 카카오톡, 키오스크 교육, 사진 정리 |
| 청년에게 | "누군가에게 필요한 사람" 경험 → 자존감 회복 |
| 어르신에게 | 디지털 소외 해소 |
| 현실성 | ★★★★★ (기술 불필요, 바로 시작 가능) |

### 모델 3: "게임 메이트" — 기억 고스톱 진행자

| 항목 | 내용 |
|------|------|
| 구조 | 요양병원 방문, 주 1-2회 |
| 활동 | 기억 고스톱, 화투 짝 맞추기 진행 |
| 청년 역할 | 게임 딜러 + 대화 촉진자 + 기록자 |
| 특징 | 부담 최소 (정해진 역할, 정해진 시간, 정해진 활동) |

### 모델 4: "이야기 채집단" — 어르신 구술 기록

| 항목 | 내용 |
|------|------|
| 구조 | 주 1회 방문, 1시간 대화 녹음 → 정리 |
| 결과물 | "할머니의 인생 책" 소책자 (가족에게 선물) |
| 청년에게 | 글쓰기/영상 포트폴리오 + 인생 간접 경험 |
| 어르신에게 | 회상치료 + "내 이야기가 기록된다"는 존재감 |
| 확장 | 유튜브 채널 "할머니의 시대" → 콘텐츠화 |

### 모델 5: "반반하우스" — 세대 공존 주거

| 항목 | 내용 |
|------|------|
| 구조 | 독거 어르신 집에 청년 입주, 월세 50% 할인 |
| 조건 | 대신 주 5시간 돌봄 (식사, 안부, 간단한 집안일) |
| 벤치마킹 | 네덜란드 **Humanitas**: 요양원에 대학생 무료 입주, 월 30시간 교류 |

## 8.3 단계적 사회 복귀 설계

**핵심: "봉사하라"가 아니라 "천천히 나와도 괜찮아"**

```
STEP 0: 비대면 참여
→ AI 안부 챗봇 모니터링 (집에서, 하루 30분)

STEP 1: 가벼운 대면
→ 주 1회 경로당 방문, 게임 진행 (2시간)

STEP 2: 정기적 관계
→ "동네 손주" 매칭, 주 2회 방문

STEP 3: 역할 확장
→ 디지털 교육, 이야기 채집, 프로그램 기획

STEP 4: 사회 복귀
→ 이 경험 기반 사회서비스/복지 분야 취업 연결
```

| AI 안부 챗봇 | 미응답 시 확인 모니터링 | 비대면 참여 (STEP 0) |
| 기억 고스톱 | 요양시설 방문 진행자 | 가벼운 대면 (STEP 1) |
| 동네 손주 | 정기 방문 매칭 | 관계 형성 (STEP 2) |
